# Moth-Eye Antireflection Fabrication Methods Lattice

MIT License - Eternally-Thriving-Grandmasterism

1. Nanoimprint Lithography (NIL) – Primary Scalable Method
   - UV-NIL / Roll-to-roll: Dexerials, Sharp MOSMITE, GEOMATEC g.moth™
   - Master: Si or Ni shim → UV resin imprint → demold
   - Throughput: m²/min scale
   - Cost at volume: <$1/m²

2. Colloidal Lithography (Nanosphere Self-Assembly)
   - PS/SiO₂ spheres → hexagonal mask → RIE etch
   - Lab-scale, rigid substrates

3. Laser Interference Lithography (LIL)
   - Two-beam interference → photoresist → etch
   - Maskless, tunable, but slower

4. Plasma / RIE on Textured Substrates
   - Black silicon base → controlled tapering
   - Semiconductor fab compatible

5. Sol-Gel Templating
   - Dip-coat + imprint → cure → etch
   - Flexible, low-cost prototypes

Recommended for MercyOS/NEXi:
- Roll-to-roll UV-NIL on PET/PC
- Combine with lotus-inspired hydrophobic topcoat for self-cleaning AR
- Valence: Low-energy, recyclable masters, no persistent chemicals
